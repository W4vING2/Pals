# Pals

Мессенджер для общения с друзьями. Чаты, звонки, посты — всё в одном месте.

## Возможности

- **Чаты** — личные и групповые сообщения в реальном времени, отправка медиа и файлов
- **Звонки** — аудио- и видеозвонки
- **Посты** — лента публикаций, возможность делиться контентом
- **Профили** — персональные страницы пользователей

## Платформы

| Платформа | Способ установки                                               |
| --------- | -------------------------------------------------------------- |
| Android   | `.apk` из [Releases](https://github.com/W4vING2/Pals/releases) |
| iOS / Web | PWA — [pals-rho.vercel.app](https://pals-rho.vercel.app)       |

## Стек

- **Frontend** — Next.js, TypeScript, Tailwind CSS
- **Backend** — Supabase (БД, авторизация, realtime)
- **Android** — Capacitor
- **Деплой** — Vercel
